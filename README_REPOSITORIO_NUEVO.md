## Pasos

1. Crear repositorio nuevo en GitHub.
2. Subir todos los archivos de este ZIP descomprimido.
3. Crear proyecto nuevo en Vercel.
4. Conectar el repositorio GPSRUTA-LUXURY.
5. Agregar las variables de entorno.
6. Deploy.
7. Probar:
   - Crear cliente
   - Crear factura
   - Adjuntar PDF
   - Enviar WhatsApp
   - Enviar correo
   - Descargar Excel completo
   - Abrir desde celular
